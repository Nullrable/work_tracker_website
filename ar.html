مزايا تطبيق تسجيل ساعات العمل

ميزات قوية لإدارة ساعات عملك وحساب العمل الإضافي وتصدير البيانات للرواتب.

تسجيل الدخول/الخروج

سجل دخولك وخروجك بنقرة واحدة. عرض الوقت الحالي للعملك بشكل مباشر. دعم عدة وظائف مع تمييز بالألوان.

حساب ذكي للعمل الإضافي

حساب تلقائي للعمل الإضافي مع إعدادات لـ 18 دولة تشمل الصين، الولايات المتحدة، اليابان، ألمانيا، فرنسا، المملكة المتحدة. قواعد قابلة للتخصيص.

الإحصائيات والرسوم البيانية

عرض اتجاهات الأرباح وتوزيع ساعات العمل حسب الوظيفة وتنبيهات العمل الإضافي. رسوم بيانية جميلة لفهم نمط عملك.

تصدير البيانات

صدّر سجلات وقتك إلى Excel أو CSV أو PDF. مثالي لتقديم كشوف المرتبات أو حفظ السجلات الشخصية.

إدارة الوظائف

إدارة عدة وظائف أو مناصب مع تمييز بالألوان المخصصة. سجل الوقت بشكل منفصل لكل وظيفة وعرض الإحصائيات لكل منها.

قفل التطبيق

حماية بياناتك برقم PIN أو المصادقة البيومترية (Face ID / Touch ID). حافظ على خصوصية وأمان سجلات وقتك.

شاهده في العمل

واجهة نظيفة وبديهية مصممة لتسجيل الدخول والخروج السريع وعرض الإحصائيات بسهولة.
